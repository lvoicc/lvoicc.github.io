Setting SSH Key

一月 13, 2019

###设置SSH Key

  1. 打开GitBash
  2. 输入cd ~/.ssh（看你是否有了ssh key 密钥，有了就备份）
  3. 输入ssh-keygen -t rsa -C "你的邮箱地址" 多次回车确认
    • 确认秘钥的保存路径（如果不需要改路径则直接回车）
    • 如果上一步置顶的保存路径下已经有秘钥文件，则需要确认是否覆盖（如果之前的秘钥不再需要则直接回车覆盖，如需要则手动拷贝到其他目录后再覆盖）
    • 创建密码（如果不需要密码则直接回车）
    • 确认密码
  4. 跟着上面所指示的路径（C:\Users\用户名\.ssh）中找到该文件，id_rsa文件即是你的私有密钥，id_rsa.pub是共开密钥
  5. 打开你的id_rsa.pub文件，复制下里面的内容，然后登录进去你的Git仓库
  6. 测试一下链接是否正常了，接着输入
    • ssh -T git@github.com（GitHub）
    • ssh -T git@git.coding.NET（Coding）
    • ssh -T git@git.dev.tencent.com（Tencent Cloud）
